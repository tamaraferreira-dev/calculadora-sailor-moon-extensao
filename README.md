# calculadora-sailor-moon-extensao
Uma extensão para o Google Chrome que transforma a Calculadora Mágica Sailor Moon num popup prático! Projeto de estudo desenvolvido com HTML, CSS (Tailwind) e JavaScript (Manifest V3), com efeitos de faíscas e frases motivadoras do anime para fazer contas rápidas de forma divertida enquanto navega na web. 🌙✨
